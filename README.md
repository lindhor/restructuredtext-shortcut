# ReStructuredText Shortcut Support for Visual Studio Code

## Keyboard Shortcuts

| Key                                               | Command                      |
| ------------------------------------------------- | ---------------------------- |
| <kbd>Ctrl</kbd> + <kbd>B</kbd>                    | Toggle bold                  |
| <kbd>Ctrl</kbd> + <kbd>I</kbd>                    | Toggle italic                |
| <kbd>Ctrl</kbd> + <kbd>`</kbd>                    | Inline Literal               |
| <kbd>Ctrl</kbd> + <kbd>L</kbd>                    | URL Link                     |
